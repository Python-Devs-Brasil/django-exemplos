Página de template

Para utilizar o sistema de templates do Djago basta criar uma página HTML básica com a estrutura padrão que será exibida em todas as páginas, por exemplo uma página com cabeçalho e rodapé, e adicionar o trecho de código abaixo no local que o conteúdo será alterado. 

            
                {% block conteudo %}{% endblock %}

Com o template definido basta criar as demais páginas que herdará este template. Para isso adicione no topo de cada página criada o código {% extends 'template/base.html' %}, definido assim o template que será herdado. Para o template saber qual conteúdo será mostrado adicione logo após o extends o trecho de código abaixo com o conteúdo da página sem adicionar o cabeçalho e o rodapé novamente. 

            
                {% block conteudo %}
                    Conteúdo da página
                {% endblock %}
