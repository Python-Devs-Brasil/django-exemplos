Listar itens do Array

Mostrar dados da lista = {'key1': 'Reposta 01', 'key2': 'Reposta 02', 'key3': 'Reposta 03', 'key4': 'Reposta 04'}

Valor da key2:

No Template adicione:

{{ dados.key2 }}

Retorna:

{{ dados.key2 }}

Exibir todos os valores

No Template adicione:

{% for key, value in dados.items %}
Key: {{ key }}
Valor: {{ value }}
{% endfor %}

Retorna:

{% for key, value in dados.items %} Key: {{ key }}
Valor: {{ value }}

{% endfor %}
