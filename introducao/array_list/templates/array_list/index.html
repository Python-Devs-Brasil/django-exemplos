Array List

Objetivo:

Mostrar na página HTML uma lista ou um dado especifico de um Array List (Dicionário)

Exemplo de Dicionário

dados = {'key1': 'Reposta 01', 'key2': 'Reposta 02', 'key3': 'Reposta 03', 'key4': 'Reposta 04'}

Exibir apenas um valor especifico (ex.: da key2):

No Template adicione:

{{ dados.key2 }}

Retorna: {{ dados.key2 }}

Exibir todos os valores como uma lista de dados

No Template adicione:

{% for key, value in dados.items %}
Key: {{ key }}
Valor: {{ value }}
{% endfor %}

Retorna:

{% for key, value in dados.items %} Key: {{ key }}
Valor: {{ value }}

{% endfor %}
